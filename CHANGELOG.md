# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-01

### Added

- Open-source governance baseline: MIT license, contributing guide, code of conduct, security policy, and support guide.
- GitHub community templates for issues and pull requests.
- CI workflow with lint, unit tests, and Playwright smoke e2e checks.
- DCO check workflow for signed-off commits.

### Changed

- Standardized e2e testing on Playwright.
- Updated project documentation for public contributor onboarding.
